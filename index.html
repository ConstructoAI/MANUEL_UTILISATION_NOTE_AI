🎯 Qu'est-ce que NOTE AI ?

NOTE AI est un assistant intelligent spécialement conçu pour les professionnels de la construction au Québec. Il utilise l'intelligence artificielle Claude Opus 4 d'Anthropic pour :

🔍 Analyser automatiquement

Vos documents de chantier avec une compréhension contextuelle avancée

📷 Interpréter les photos

De construction avec expertise technique spécialisée

📄 Générer des rapports

Professionnels conformes aux normes québécoises

📋 Organiser vos notes

Par catégories de chantier pour un suivi optimal

✅ Assurer la conformité

Avec les exigences CCQ, RBQ et CNESST

À qui s'adresse NOTE AI ?

  • Entrepreneurs généraux : Coordination globale de projets
  • Sous-traitants spécialisés : Rapports métier spécifiques
  • Gestionnaires de projet : Suivi multi-chantiers
  • Inspecteurs et surveillants : Documentation de conformité
  • Chargés de projet : Communication client-équipe

Avantages clés

  • ✅ Spécialisé construction québécoise : Terminologie et normes intégrées
  • ✅ Analyse IA avancée : Compréhension contextuelle des documents
  • ✅ Interface intuitive : Optimisée pour utilisation terrain
  • ✅ Rapports professionnels : Conformes aux standards de l'industrie
  • ✅ Sécurité : Données traitées localement, confidentialité garantie

⚙️ Installation et configuration

Étape 1 : Prérequis

  • Python 3.8+ installé sur votre système
  • Connexion Internet pour télécharger les dépendances
  • Clé API Claude d'Anthropic

Étape 2 : Installation automatique

  1. Téléchargez tous les fichiers du projet dans un dossier
  2. Double-cliquez sur Run.bat
  3. Le script détecte automatiquement Python et installe les dépendances
  4. L'interface s'ouvre automatiquement dans votre navigateur

Étape 3 : Installation manuelle (si nécessaire)

# Installer les dépendances
pip install -r requirements.txt

# Lancer l'application
streamlit run app.py

Configuration API Claude

Obtenir votre clé API

  1. Visitez console.anthropic.com
  2. Créez un compte ou connectez-vous
  3. Allez dans "API Keys"
  4. Cliquez "Create Key"
  5. Copiez la clé complète (commence par sk-ant-api03-)

Méthodes de configuration

Option 1 : Interface

  • Lancez NOTE AI
  • Barre latérale → "🔑 Configuration API"
  • Collez votre clé
  • Cliquez "💾 Configurer"

Option 2 : Variable d'environnement

# Windows
set ANTHROPIC_API_KEY=sk-ant-api03-votre-clé-ici

# Linux/Mac
export ANTHROPIC_API_KEY=sk-ant-api03-votre-clé-ici

Option 3 : Fichier .env

Créez un fichier .env dans le dossier du projet :

ANTHROPIC_API_KEY=sk-ant-api03-votre-clé-ici

Déploiement Hugging Face Spaces

NOTE AI est prêt pour le déploiement sur Hugging Face Spaces :

  1. Créez un Space sur huggingface.co/spaces
  2. Sélectionnez Streamlit comme SDK
  3. Uploadez tous les fichiers du projet
  4. Ajoutez votre clé API dans Settings → Secrets :
    • Nom : ANTHROPIC_API_KEY
    • Valeur : votre clé API complète

✅ Bonnes pratiques

Prise de notes efficace

Organisation quotidienne

  1. Commencez tôt : Première note dès l'arrivée sur chantier
  2. Soyez régulier : Notes toutes les 2-3 heures minimum
  3. Contextualisez : Toujours préciser lieu et responsable
  4. Photographiez : Une image vaut mille mots

Catégorisation

  • Utilisez les bonnes catégories : Facilite la génération de rapports
  • Soyez spécifique : "Avancement des travaux" plutôt qu'"Autre"
  • Détaillez la localisation : "2e étage, appartement 204" vs "2e étage"

Contenu des notes

✅ BONNE PRATIQUE :
Catégorie : Avancement des travaux Note : Coulée de béton dalle 2e étage terminée à 14h30 Volume : 85 m³, résistance 25 MPa Température extérieure : -8°C Protection thermique installée (bâches chauffantes) Cure débutera demain matin si température stable Localisation : 2e étage, dalle principale Responsable : Pierre Lavoie, contremaître béton
❌ À ÉVITER :
Catégorie : Autre Note : Béton coulé Localisation : 2e Responsable : Pierre

Gestion des documents

Upload optimal

  • Nommage : Utilisez des noms explicites
    • ✅ "Plan_electrique_2e_etage_rev3.pdf"
    • ❌ "plan.pdf"
  • Qualité : Privilégiez la haute résolution pour les images
  • Format : Utilisez PDF pour documents officiels
  • Taille : Compressez si > 50 MB pour accélérer l'analyse

Types de photos recommandées

  1. Vues d'ensemble : Contexte général de la zone
  2. Détails techniques : Assemblages, connections
  3. Problèmes : Non-conformités, défauts
  4. Avancement : Comparaison avant/après
  5. Sécurité : Équipements, signalisation

🔧 Dépannage

Problèmes de configuration API

❌ "Client Anthropic non disponible"

Cause : Clé API non configurée ou invalide

Solutions :

  1. Vérifiez que votre clé commence par sk-ant-api03-
  2. Testez la clé sur console.anthropic.com
  3. Utilisez le bouton "🧪 Test Direct (Méthode EXPERTS IA)"
  4. Redémarrez l'application après configuration

❌ "Connexion API échouée"

Cause : Problème réseau ou proxy

Solutions :

  1. Vérifiez votre connexion Internet
  2. Testez l'accès à https://api.anthropic.com
  3. Configurez le proxy si nécessaire
  4. Utilisez le bouton "🔧 Debug Système" pour diagnostic

Problèmes d'installation

❌ Python non trouvé

Cause : Python non installé ou pas dans le PATH

Solutions :

  1. Installez Python depuis python.org
  2. Cochez "Add Python to PATH" durant l'installation
  3. Redémarrez votre ordinateur
  4. Désactivez les alias Microsoft Store pour Python

❌ Dépendances manquantes

Cause : Installation incomplète des packages

Solutions :

# Mise à jour pip
python -m pip install --upgrade pip

# Installation forcée
pip install -r requirements.txt --force-reinstall

# Installation individuelle si problème
pip install streamlit anthropic Pillow python-dotenv PyPDF2 pandas python-docx openpyxl

Codes d'erreur courants

Code Message Cause Solution
API_001 "Client Anthropic non disponible" Clé API manquante Configurer la clé API
API_002 "Erreur d'authentification" Clé API invalide Vérifier la clé API
FILE_001 "Fichier trop volumineux" Taille > 200 MB Compresser ou diviser
FILE_002 "Format non supporté" Extension non reconnue Convertir au bon format
