⚙️Installation et configuration

💻Installation locale (Windows)

1️⃣Prérequis

  • Python 3.8+ installé sur votre système
  • Connexion Internet pour télécharger les dépendances
  • Clé API Claude d'Anthropic

2️⃣Installation automatique

  1. Téléchargez tous les fichiers du projet dans un dossier
  2. Double-cliquez sur Run.bat
  3. Le script détecte automatiquement Python et installe les dépendances
  4. L'interface s'ouvre automatiquement dans votre navigateur

3️⃣Installation manuelle (si nécessaire)

# Installer les dépendances pip install -r requirements.txt # Lancer l'application streamlit run app.py

🔑Configuration API Claude

🌐Obtenir votre clé API

  1. Visitez console.anthropic.com
  2. Créez un compte ou connectez-vous
  3. Allez dans "API Keys"
  4. Cliquez "Create Key"
  5. Copiez la clé complète (commence par sk-ant-api03-)

⚙️Méthodes de configuration

Option 1 : Configuration dans l'interface

  • Lancez NOTE AI
  • Dans la barre latérale, section "🔑 Configuration API"
  • Collez votre clé dans le champ "Clé API Anthropic"
  • Cliquez "💾 Configurer"

🚀Déploiement Hugging Face Spaces

NOTE AI est prêt pour le déploiement sur Hugging Face Spaces :

  1. Créez un Space sur huggingface.co/spaces
  2. Sélectionnez Streamlit comme SDK
  3. Uploadez tous les fichiers du projet
  4. Ajoutez votre clé API dans les Settings → Secrets

🔧Dépannage

❌Problèmes de configuration API

🚫Erreur : "Client Anthropic non disponible"

Cause : Clé API non configurée ou invalide

Solutions :

  1. Vérifiez que votre clé commence par sk-ant-api03-
  2. Testez la clé sur console.anthropic.com
  3. Utilisez le bouton "🧪 Test Direct (Méthode EXPERTS IA)"
  4. Redémarrez l'application après configuration

🌐Erreur : "Connexion API échouée"

Cause : Problème réseau ou proxy

Solutions :

  1. Vérifiez votre connexion Internet
  2. Testez l'accès à https://api.anthropic.com
  3. Configurez le proxy si nécessaire
  4. Utilisez le bouton "🔧 Debug Système" pour diagnostic

🐍Problèmes d'installation

❌Python non trouvé

Cause : Python non installé ou pas dans le PATH

Solutions :

  1. Installez Python depuis python.org
  2. Cochez "Add Python to PATH" durant l'installation
  3. Redémarrez votre ordinateur
  4. Désactivez les alias Microsoft Store pour Python

📦Dépendances manquantes

Cause : Installation incomplète des packages

Solutions :

# Mise à jour pip python -m pip install --upgrade pip # Installation forcée pip install -r requirements.txt --force-reinstall # Installation individuelle si problème pip install streamlit anthropic Pillow python-dotenv PyPDF2 pandas python-docx openpyxl

📄Problèmes d'analyse de fichiers

📋"PDF vide ou impossible à lire"

Cause : PDF scanné ou protégé

Solutions :

  1. Utilisez un PDF avec texte sélectionnable
  2. Convertissez les PDF scannés avec OCR
  3. Exportez depuis le logiciel original si possible

🖼️"Erreur d'analyse d'image"

Cause : Format non supporté ou fichier corrompu

Solutions :

  1. Convertissez en JPG ou PNG
  2. Réduisez la taille si > 200 MB
  3. Vérifiez que l'image n'est pas corrompue

🔧Diagnostic système

Utilisez le bouton "🔧 Debug Système" pour obtenir :

  • Versions des modules Python installés
  • Connectivité aux serveurs Anthropic
  • Configuration des proxies et DNS
  • Fichiers de configuration présents
